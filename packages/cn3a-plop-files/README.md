# create-nuxt3-app

See more information about usage on the main [ReadMe](../../README.md).

## Development

1. Install dependencies.
``npm install``
2. Run plop script from project root directory.
``npm run plop``

## Folder structure

``/plops``
In this folder, you'll find the different generator configs.

``/plop-template``
You'll find the templates that we use to generate our new files
